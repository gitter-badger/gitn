# gitn
Gitn is [Denite.nvim][denite] source and kind set for using git

# Requirements
## Plugins
- [Denite.nvim][denite]

# Features
## gitn(not implemented)
Show implemented commands of gitn

## gitn:status
Show `git status` results

## gitn:log
Show `git log` results
